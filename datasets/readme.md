# Zbiory Danych (Datasets)

Katalog ten przeznaczony jest na dane służące do treningu oraz ewaluacji modeli.

## Pobieranie Danych

1. Przejdź pod adres: [LINK DO DATASETÓW]("")
2. Pobierz plik `datasets.zip`.
3. Rozpakuj archiwum w katalogu `datasets/`.

## Struktura Katalogów
Po wypakowaniu struktura powinna wyglądać następująco:

- `datasets/dataset_classification/` - dane do klasyfikacji
- `datasets/dataset_entry/` - dane dla Kontrolera Domeny
- `datasets/dataset_segmentation/` - dane dla modeli segmentacyjnych
