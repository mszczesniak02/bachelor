# Modele Sieci Neuronowych

Aby system funkcjonował poprawnie, wymagane jest pobranie wyuczonych wag modeli.

## Instrukcja Pobierania

1. Przejdź na stronę: [LINK DO MODELI](https://drive.google.com/file/d/1VSe1yXOsGb3sahzwu3HNdIBED9SaQfEN/view?usp=sharing)
2. Pobierz plik `models.zip`.
3. Rozpakuj zawartość archiwum bezpośrednio do katalogu `models/`.

## Wymagana Struktura
Po rozpakowaniu upewnij się, że pliki modeli znajdują się bezpośrednio w odpowiednich podfolderach lub głównej ścieżce, zgodnie z oczekiwaniami skryptów inferencyjnych (np. `models/model.pth`).
